Текущий пользователь: Сменить пользователя Перейти к таблицам

Данная программа позволяет удобно работать с базой данных сети магазинов одежды.

Пользователь может выполнить вход в свою учетную запись или работать анонимно. Для администраторов базы данных предусмотрены возможности редактирования данных. Для обычных пользователей эти возможности ограничены. Анонимные пользователи могут только просматривать содержимое таблиц.

С помощью панель сверху каждой таблицы расположены кнопки взаимодействия с данными текущей таблицы. В поле поиска вы можете выбрать столбец таблицы, по которому будет происходить поиск. Столбец выбирается в выпадающем списке.

Добавление новой строки осуществляется путем нажатия на кнопку "Добавить". Необходимо ввести поля строки для ее добавления. Id генерируется автоматически. Редактирование строки производится нажатием на ее колонку, после завершения редактирования необходимо нажать на кнопку "Сохранить изменения". Удалить строку из таблицы можно при нажатии на соответствующую кнопку в удаляемой строке. Также возможно удалить все строки из таблицы стразу, нажав на красную кнопку "Удалить все" (не рекомендуется).

Для администраторов существует отдельная страница, где можно просматривать некоторые данные. Попасть на нее можно при нажатии на ссылку "Панель администратора" в меню сверху страницы.

С помощью комбинации клавиш ctrl+p (cmd+p) вы можете распечатать данные. Интерфейс приложения будет упрощен
